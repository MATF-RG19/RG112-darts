| Controls | Description |
| --- | --- |
| LMB | throws dart (HOLD) |
| RMB | resets the game |
| V | changes the shade model |
| R | randomizes the colour of the dart |
| G | bad boy amongs all R's |
| F | inspects the dart (TOGGLE) |
